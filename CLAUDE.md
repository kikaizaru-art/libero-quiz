# リベロクイズ — 開発メモ

ビルド不要の静的Webアプリ(PWA)。`js/data.js` がクイズデータ、`js/app.js` がアプリロジック。

## 難易度(ステージ)の定義 — 必ず守ること

各分野の3ステージは学校教育の段階が基準。問題の追加・変更時は必ずこの定義で振り分ける:

- **初級** = 中学生までに学ぶ内容(小・中学校の教科書レベル/一般常識として広く知られる事項)
- **中級** = 高校で学ぶ内容(高校の教科書・資料集レベル)
- **上級** = 大学レベル(高校の範囲を超える大学教養・専門レベル)

補足ルール:

- 同じレベル帯に候補が多い場合は、より馴染み深いものを下のステージに置く(各ステージ5問を維持)
- 判断基準は「その事項がどの学校段階の教科書・資料集に載るか」。問題文の言い回しではなく問われている知識で判定する

## その他の注意

- `js/data.js` を変更したら `sw.js` の `CACHE` バージョンを上げる(PWAキャッシュ更新のため)
- 進捗は localStorage に `catId:stageIdx:qIdx` 形式のキーで保存されるため、既存問題の並べ替えは既存ユーザーの復習リスト・ライブラリ解放状態とずれる点に留意
