# リベロクイズ — 1日5問のビジネス教養

ビジネスパーソンがリベラルアーツ(教養)を継続的に学ぶためのクイズアプリです。
ステージ制・レベル・デイリー目標・実績といった継続を促す仕組みを取り入れつつ、落ち着いたトーンのUIで、すき間時間の学習習慣づくりを支援します。

## 使い方

ビルド不要の静的Webアプリです。`index.html` をブラウザで開くだけで利用できます。

```bash
# ローカルサーバーで起動する場合
python3 -m http.server 8000
# → http://localhost:8000 をスマホ/PCのブラウザで開く
```

## デプロイ(Vercel)

Vercelにインポート済みであれば、ブランチにプッシュするだけで自動デプロイされます。

- **本番**:`main` ブランチへのマージで本番URLが更新
- **プレビュー**:作業ブランチへのプッシュごとにプレビューURLが自動発行

初回のみ [vercel.com/new](https://vercel.com/new) でこのリポジトリをインポートしてください(静的サイトなのでビルド設定は不要、そのまま Deploy でOK)。`vercel.json` に基本設定を同梱しています。GitHub Pages でも同様に公開可能です。

## 学習システム(継続を促す仕組み)

| 要素 | 内容 |
|---|---|
| 今日の5問 | ホームの主ボタン。日替わりで5つの分野から1問ずつ出題(ステージ進捗とは独立)。その日の初回クリアでボーナスXP。クリア後は明日の出題分野を予告 |
| 今週の学習 | ホーム上部に今週7日分のドットを表示。学習した日が点灯し「鎖を切りたくない」動機を作る |
| 学習を進める | ホームのサブカード。続きのステージを自動選択して1タップで学習再開 |
| ステージ制 | 6分野 × 3ステージ(初級→中級→上級)。クリアで次が解放 |
| 星3評価 | 8問中 5問=★ / 7問=★★ / 全問=★★★(6割正解でクリア)。全ステージ星3が到達目標 |
| XP・レベル | 正解でXP獲得。連続正解でボーナスXP |
| 本日の目標 | 毎日3つの目標(1クリア/10問正解/3問連続正解)で報酬XP。ホームで1行サマリー表示 |
| 連続学習日数 | 学習した日数の連続記録。今日学習するまでグレー、学習で点灯表示。3〜100日の各節目で実績解除 |
| ストリークフリーズ | 7日連続ごとに1個獲得(最大2個ストック)。1日休んでも自動消費して連続記録を守る |
| 復習タブ | 間違えた問題を自動で記録し、最大5問ずつ再挑戦。正解すればリストから消える(克服)。待ち件数はナビにバッジ表示 |
| 弱点特訓 | 正答率が低い分野(10問以上解答・正答率85%未満)をホームの学習メニューで提案し、その分野から集中的に8問出題 |
| ○×スピード | 全分野からランダム10問を「答えは◯◯である」の正誤判定で出題。1問10秒の制限時間バー付きで、同じ問題をテンポよく解き直せる |
| カード当て | 解説文(正解の語は伏せ字)からライブラリの知識カードを4択で当てる逆引きクイズ。解放済みカード8枚で挑戦可能 |
| 定着チェック | 正解した問題も忘れた頃(14日→30日→60日、以降60日ごと)に復習へ戻して抜き打ち確認。間違えたら通常の復習リストで鍛え直す |
| ふりかえり | リザルトで間違えた問題と正解を一覧表示し、復習リストへの追加を明示 |
| 記録タブ | 学習サマリー、月別の学習カレンダー、分野別正答率に加え、実績(全25種)をセグメント切替で表示 |
| 週間レポート | 記録タブに今週(日曜始まり)の解答数・正答率・新カード数・学習日数を表示し、先週との比較を一言で添える |
| ピン留め | 知識カードを開いて「ピン留めする」を押すと、あとで語りたいカードに目印を付けられる。ライブラリの「ピン留め」ピルで一覧表示 |
| 結果共有 | リザルトから成績カード画像(ランク・星・連続日数入り)を生成してSNS等に共有。共有シート未対応の環境では画像を保存 |
| PWA | ホーム画面に追加してアプリのように起動可能。Service Workerによるオフライン対応 |
| 自動保存 | 進捗は localStorage に自動保存(端末内で完結) |

下部ナビは「ホーム / 学習 / 復習 / 記録」の4タブ構成です。ダーク/ライトモードはOSの設定に自動追従します。

## 難易度(ステージ)の定義

各分野の3ステージは、学校教育の段階を基準に次のように定義しています。

| ステージ | 基準 |
|---|---|
| 初級 | 中学生までに学ぶ内容(小・中学校の教科書レベル/一般常識として広く知られる事項) |
| 中級 | 高校で学ぶ内容(高校の教科書・資料集レベル) |
| 上級 | 大学レベル(高校の範囲を超える大学教養・専門レベル) |

問題を追加するときも必ずこの基準でステージを選びます。同じレベル帯に候補が多い場合は、より馴染み深いものを下のステージに置きます。

## 収録分野(全144問・全問解説付き)

- 哲学 — ソクラテスからロールズまで
- 歴史 — 古代エジプトから冷戦終結まで
- 芸術 — ルネサンスから現代アートまで
- 科学 — ニュートンから量子力学まで
- 文学 — 源氏物語からマジックリアリズムまで
- 音楽 — グレゴリオ聖歌から十二音技法まで

すべての問題に「解説」と、内容を象徴するイラスト(インラインSVG)が付いており、正解しても間違えても知識が深まり、視覚的にも記憶に残る設計です。

## 構成

```
index.html      # 画面構造(SPA)
css/style.css   # UIスタイル(モバイルファースト、ダーク/ライト対応)
js/data.js      # クイズデータ(分野→ステージ→問題)
js/art.js       # 知識カードのイラスト(lib.title → インラインSVG)
js/app.js       # アプリロジック(XP/目標/実績/保存)
manifest.json   # PWAマニフェスト
sw.js           # Service Worker(オフラインキャッシュ)
icon.svg        # アプリアイコン
docs/           # 設計ドキュメント(UI/UX再構成プラン)
```

## 問題の増やし方

`js/data.js` の該当分野の `questions` 配列にオブジェクトを追加するだけです。追加先のステージは上記「難易度の定義」(初級=中学まで/中級=高校/上級=大学レベル)に従って選んでください。

```js
{
  q: "問題文",
  choices: ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  answer: 0,        // 正解のインデックス(0始まり)
  exp: "解説文"
}
```

分野の追加も `QUIZ_DATA` に同じ形式のオブジェクトを足せばOK(実績は自動生成されます)。

問題を追加したら、`js/art.js` の `QUIZ_ART` に同じ `lib.title` をキーにしたイラストSVGも追加します(イラストがない項目は表示が自動でスキップされます)。描き方の決まりは `js/art.js` 冒頭のコメントを参照してください。
