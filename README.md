# 🎓 リベロクイズ — 遊んで深める教養

リベラルアーツ(教養)を楽しく継続的に学ぶためのクイズアプリです。
スマホゲームの設計(ステージ制・レベル・デイリーミッション・実績)を取り入れ、「やる気が出て続けたくなる」ことを最優先にデザインしています。

## 🚀 遊び方

ビルド不要の静的Webアプリです。`index.html` をブラウザで開くだけで遊べます。

```bash
# ローカルサーバーで起動する場合
python3 -m http.server 8000
# → http://localhost:8000 をスマホ/PCのブラウザで開く
```

GitHub Pages にそのままデプロイ可能です(Settings → Pages → ブランチを選択)。

## 🎮 ゲームシステム(スマホゲーム設計の採用要素)

| 要素 | 内容 |
|---|---|
| 🗺️ ステージ制 | 6分野 × 3ステージ(初級→中級→上級)。クリアで次が解放 |
| ⭐ 星3評価 | 5問中 3問=★ / 4問=★★ / 全問=★★★。やり込みで全ステージ星3を目指す |
| ⚡ XP・レベル | 正解でXP獲得、レベルアップ演出付き。コンボでボーナスXP |
| 🔥 コンボ | 連続正解でコンボ数に応じてXPが増える |
| 📋 デイリーミッション | 毎日3つのミッション(1クリア/10問正解/3コンボ)で報酬XP |
| 🔥 ストリーク | 連続プレイ日数を記録。3日・7日で実績解除 |
| 🏅 実績バッジ | 全15種。分野マスター、パーフェクト、継続系など |
| 💾 オート保存 | 進捗は localStorage に自動保存(端末内で完結) |

## 📚 収録分野(全90問・全問解説付き)

- 🦉 哲学 — ソクラテスからロールズまで
- 🏛️ 歴史 — 古代エジプトから冷戦終結まで
- 🎨 芸術 — ルネサンスから現代アートまで
- 🔭 科学 — ニュートンから量子力学まで
- 📚 文学 — 源氏物語からマジックリアリズムまで
- 🎼 音楽 — グレゴリオ聖歌から十二音技法まで

すべての問題に「解説」が付いており、正解しても間違えても知識が深まる設計です。

## 🗂️ 構成

```
index.html      # 画面構造(SPA)
css/style.css   # スマホゲーム風UI(モバイルファースト)
js/data.js      # クイズデータ(分野→ステージ→問題)
js/app.js       # ゲームロジック(XP/ミッション/実績/保存)
```

## ➕ 問題の増やし方

`js/data.js` の該当分野の `questions` 配列にオブジェクトを追加するだけです。

```js
{
  q: "問題文",
  choices: ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  answer: 0,        // 正解のインデックス(0始まり)
  exp: "解説文"
}
```

分野の追加も `QUIZ_DATA` に同じ形式のオブジェクトを足せばOK(実績バッジは自動生成されます)。
