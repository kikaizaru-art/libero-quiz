# リベロクイズ — 1日5問のビジネス教養

ビジネスパーソンがリベラルアーツ(教養)を継続的に学ぶためのクイズアプリです。
ステージ制・レベル・デイリー目標・実績といった継続を促す仕組みを取り入れつつ、落ち着いたトーンのUIで、すき間時間の学習習慣づくりを支援します。

## 使い方

ビルド不要の静的Webアプリです。`index.html` をブラウザで開くだけで利用できます。

```bash
# ローカルサーバーで起動する場合
python3 -m http.server 8000
# → http://localhost:8000 をスマホ/PCのブラウザで開く
```

## デプロイ(Vercel)

Vercelにインポート済みであれば、ブランチにプッシュするだけで自動デプロイされます。

- **本番**:`main` ブランチへのマージで本番URLが更新
- **プレビュー**:作業ブランチへのプッシュごとにプレビューURLが自動発行

初回のみ [vercel.com/new](https://vercel.com/new) でこのリポジトリをインポートしてください(静的サイトなのでビルド設定は不要、そのまま Deploy でOK)。`vercel.json` に基本設定を同梱しています。GitHub Pages でも同様に公開可能です。

## 学習システム(継続を促す仕組み)

| 要素 | 内容 |
|---|---|
| ステージ制 | 6分野 × 3ステージ(初級→中級→上級)。クリアで次が解放 |
| 星3評価 | 5問中 3問=★ / 4問=★★ / 全問=★★★。全ステージ星3が到達目標 |
| XP・レベル | 正解でXP獲得。連続正解でボーナスXP |
| 連続正解 | 連続で正解するほどXPが増える |
| 本日の目標 | 毎日3つの目標(1クリア/10問正解/3問連続正解)で報酬XP |
| 連続学習日数 | 学習した日数の連続記録。3日・7日で実績解除 |
| 復習モード | 間違えた問題を自動で記録し、ホームから最大5問ずつ再挑戦。正解すればリストから消える(克服) |
| 記録画面 | 学習日数・正答率などのサマリー、月別の学習カレンダー(タブ切替)、分野別正答率を表示 |
| 実績 | 全16種。分野マスター、全問正解、弱点克服、継続系など |
| 自動保存 | 進捗は localStorage に自動保存(端末内で完結) |

## 収録分野(全90問・全問解説付き)

- 哲学 — ソクラテスからロールズまで
- 歴史 — 古代エジプトから冷戦終結まで
- 芸術 — ルネサンスから現代アートまで
- 科学 — ニュートンから量子力学まで
- 文学 — 源氏物語からマジックリアリズムまで
- 音楽 — グレゴリオ聖歌から十二音技法まで

すべての問題に「解説」が付いており、正解しても間違えても知識が深まる設計です。

## 構成

```
index.html      # 画面構造(SPA)
css/style.css   # UIスタイル(モバイルファースト)
js/data.js      # クイズデータ(分野→ステージ→問題)
js/app.js       # アプリロジック(XP/目標/実績/保存)
```

## 問題の増やし方

`js/data.js` の該当分野の `questions` 配列にオブジェクトを追加するだけです。

```js
{
  q: "問題文",
  choices: ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  answer: 0,        // 正解のインデックス(0始まり)
  exp: "解説文"
}
```

分野の追加も `QUIZ_DATA` に同じ形式のオブジェクトを足せばOK(実績は自動生成されます)。
